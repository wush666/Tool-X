# Tool-X
## 法律声明：请勿使用此工具
* 据那个人说，这个工具有害，可能会损坏你的电脑，所以我请求大家不要安装和使用这个工具。

### 引言
* Tool-X是针对Termux和Linux系统的Kali Linux黑客工具安装器。Tool-X是为Termux和基于Linux的系统开发的。使用Tool-X，您可以在Termux（安卓）和其他基于Linux的发行版中安装近370多种黑客工具。现在，Tool-X已可用于Ubuntu、Debian等系统。

### 操作系统要求
* Tool-X可在以下任何操作系统上运行：<br>
* Android（使用Termux应用）<br/>
* Linux（基于Debian的系统）<br/>
* Unix
------------------------------------------------------------------------
### 如何安装
打开终端并输入以下命令。
* `apt update`
* `apt install git`
* `git clone https://github.com/wush666/Tool-X.git`
* `cd Tool-X`
* `chmod +x install`
* `sh install` 如果安装不成功，请使用 `./install`
------------------------------------------------------------------------
### 如何使用？
输入一个数字以获取特定输出：

* (1)：显示所有可用工具，并输入您想要安装的工具编号。
* (2)：显示工具类别。
* (3)：如果您想更新Tool-X。
* (4)：如果您想了解我们。
* (x)：退出工具。

在终端的任何位置输入 Tool-X 或 toolx，即可打开 Tool-X。
